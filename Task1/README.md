# 🎯 Number Guessing Game

## 📌 Description

This is a simple Java-based game where the user has to guess a randomly generated number within a given range. The program provides hints to guide the user toward the correct answer.

---

## 🚀 Features

* Random number generation
* User input handling
* Hints (Too High / Too Low)
* Limited attempts *(optional)*
* Score tracking *(optional)*

---

## 🛠️ Technologies Used

* Java
* Scanner Class
* Random Class

---

## ▶️ How to Run

```bash
javac NumberGuessingGame.java
java NumberGuessingGame
```

---

## 📷 Sample Output

```
Enter your guess: 50
Too high!
Enter your guess: 25
Too low!
Congratulations! You guessed the number.
```

## 💡 Concepts Used

* Loops
* Conditional Statements
* Basic Input/Output
