
# 🏧 ATM Interface

## 📌 Description

The **ATM Interface** is a Java-based application that simulates basic banking operations such as withdrawing money, depositing funds, and checking account balance.

This project demonstrates the use of object-oriented programming concepts and provides a simple menu-driven interface for user interaction.

---

## 🎯 Objectives

* To simulate real-world ATM functionalities
* To implement secure and structured user interactions
* To handle transactions like deposit and withdrawal
* To maintain and display account balance

---

## 🚀 Features

* Check account balance
* Deposit money
* Withdraw money
* Input validation (e.g., insufficient balance)
* Menu-driven interface for easy navigation

---

## 🛠️ Technologies Used

* Java ☕
* Object-Oriented Programming (OOP)

---
