
# 📊 Student Grade Calculator

## 📌 Description

The **Student Grade Calculator** is a Java-based application that calculates a student’s total marks, average percentage, and assigns a grade based on the calculated performance.

This project demonstrates basic programming concepts such as input handling, arithmetic operations, and conditional logic.

---

## 🎯 Objectives

* To calculate total marks from multiple subjects
* To compute the average percentage
* To assign grades based on performance
* To display a clear and formatted result

---

## 🚀 Features

* Accepts marks for multiple subjects
* Calculates:

  * Total Marks
  * Average Percentage
* Assigns grades (A, B, C, etc.)
* Displays result in a user-friendly format
* Handles basic input validation

---

## 🛠️ Technologies Used

* Java ☕
* Scanner Class (for input handling)

---

## ⚙️ How It Works

1. User enters marks for different subjects
2. Program calculates total marks
3. Average percentage is computed
4. Grade is assigned based on predefined criteria
5. Final result is displayed

---

## ▶️ How to Run

### 1️⃣ Compile the Program

```bash
javac GradeCalculator.java
```

### 2️⃣ Run the Program

```bash
java GradeCalculator
```

## 📚 Grading Criteria (Example)

* A → 90% and above
* B → 75% to 89%
* C → 60% to 74%
* D → 50% to 59%
* F → Below 50%

*(You can modify grading logic as per requirements)*
